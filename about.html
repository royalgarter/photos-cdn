Why Photos CDN?

Everything you need for image delivery — without the complexity.

Semantic Search

128-dimensional Gemini embeddings find the closest matching image in your database via cosine similarity — no rigid tagging required.

On-Demand Generation

No close match? AI generates a new image using Gemini → Cloudflare AI → Pollinations → HuggingFace in a provider chain. Always delivers.

Responsive srcset

One endpoint returns a complete srcset across 12 breakpoints — from 150px thumbnail to 3840×2160 4K desktop. Drop it straight into your <img> tag.

Format Conversion

Request output=webp, output=png, or output=jpg on the fly. WebP saves ~25% over JPEG with no quality loss.

Async Generation (RFC 7240)

Add Prefer: respond-async to never block a page load. Get a fallback image instantly, then poll to upgrade when the AI generation finishes.

ArangoDB Vector Index

Powered by ArangoDB APPROX_NEAR_COSINE — a real vector index, not a brute-force scan. Fast similarity search even at scale.

Cloudflare R2 Storage

Generated images are compressed via resmush.it and uploaded to Cloudflare R2 with a public CDN domain. Cache-Control: max-age=31536000 on every hit.

Genre-Aware Prompting

106 photography genres (astrophotography, fashion, wedding, ruins…) each carry a curated prompt template that enriches the AI generation request automatically.

Daily Auto-Indexer

A background worker fetches curated photos from Pexels, Unsplash, Pixabay, and Picjumbo every 24 hours, embeds them, and grows the DB automatically.

How It Works

Every request goes through a 4-phase pipeline:

01
Intercept

Parse width, height, text, category, seed, format, output from the URL.

02
Vector Search

Embed query via Gemini (128-dim). Cosine search across ArangoDB. Score ≥ 0.85 → serve cached image.

03
Generate

Score < 0.85 → apply genre template → Gemini → Cloudflare AI → Pollinations → HuggingFace. Resize to 12 variants, upload to R2.

04
Deliver

302 redirect to CDN URL for jpg. Sharp-convert for webp/png. Headers carry similarity score and cache directives.

Image Variants

Every image is generated at 12 preset resolutions, from 150×150 thumbnail to 3840×2160 4K. The server selects the closest variant to the requested dimensions:

  • desktop_4k — 3840×2160
  • desktop_1440p — 2560×1440
  • desktop_1080p — 1920×1080
  • desktop_budget — 1366×768
  • tablet_wide — 1280×800
  • tablet_standard — 768×1024
  • mobile_large — 412×915
  • mobile_medium — 390×844
  • mobile_standard — 360×800
  • original — 1024×1024
  • medium — 400×400
  • thumbnail — 150×150
